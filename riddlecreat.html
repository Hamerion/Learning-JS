Bienvenue sur le createur d'énigme!
Pour l'utilisé écrivez votre question dans l'emplacement prévue. Pour créez la réponse à cette question indiquez les mots clées que le programme devra chercher.Si vous souhaité que le programme accepte plusieurs réponses créez un nouvel emplacement avec le bouton "ajoutez mots-clez "ou" ".
Si vous souhaitez que plusieurs mots-clés apparaissent choisissez"ajoutez mots-clez "et" ".
Appuyez ensuite sur "soumettre l'énigme" pour cacher les reponses
